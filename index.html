Статус лабораторных

В ячейке две иконки: первая — оценка, вторая — PR.
Для оценок: ✅ — зачтённая оценка, 🟥 — двойка, ❌ — оценки нет. В колонке КН показывается только оценка, без PR.

Легенда

✅✅ — оценка есть, PR есть

✅❌ — оценка есть, PR не найден

🟥✅ — в журнале стоит двойка, PR есть

🟥❌ — в журнале стоит двойка, PR не найден

❌✅ — PR есть, оценки пока нет

❌❌ — ни оценки, ни PR

КН — отдельная колонка контрольной недели, там показывается только эмодзи оценки
